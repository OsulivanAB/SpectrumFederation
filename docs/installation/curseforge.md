# CurseForge Installation

CurseForge is one of the most popular addon managers for World of Warcraft.

## Prerequisites

1. Download and install [CurseForge App](https://www.curseforge.com/download/app)
2. Launch the app and ensure it detects your WoW installation

### Step 2: Install

1. Click on the **Spectrum Federation** addon
2. Click the **Install** button
3. Wait for the download and installation to complete

### Step 3: Verify Installation

1. Launch World of Warcraft
2. At the character selection screen, click **AddOns**
3. Verify that "Spectrum Federation" is listed and enabled
4. Enter the game and check for the load message in chat

## Updating

CurseForge automatically checks for updates:

1. The app will notify you when updates are available
2. Click **Update** next to Spectrum Federation
3. Or use **Update All** to update all addons at once

## Release Channels

CurseForge supports different release channels:

- **Release**: Stable versions only (default)
- **Beta**: Beta and stable versions
- **Alpha**: All versions including alpha releases

To change the release channel:

1. Right-click on Spectrum Federation in your addon list
2. Select **Change Release Type**
3. Choose your preferred channel

!!! warning "Beta/Alpha Releases"
    Beta and alpha versions may contain experimental features and bugs. Use at your own risk!

## Uninstalling

To remove the addon:

1. In the CurseForge App, find Spectrum Federation
2. Click the **Uninstall** button
3. Confirm the removal
