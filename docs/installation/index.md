# Installation Guide

There are multiple ways to install the Spectrum Federation addon. Choose the method that works best for you.

## Recommended Methods

### WowUp (Recommended)

WowUp is a modern, open-source addon manager that makes it easy to install and update addons.

[→ WowUp Installation Guide](wowup.md)

### CurseForge

CurseForge is the traditional addon manager used by many WoW players.

[→ CurseForge Installation Guide](curseforge.md)

## Alternative Method

### Manual Installation

If you prefer to install addons manually or want to install from source, you can do so directly.

[→ Manual Installation Guide](manual.md)

## After Installation

Once installed, the addon will automatically load when you start World of Warcraft. You should see a confirmation message in chat:

```
[Spectrum Federation] loaded successfully!
```

## Troubleshooting

!!! tip "Enable Script Errors"
    If you encounter issues, enable script errors in-game with:
    ```
    /console scriptErrors 1
    ```

!!! tip "Reload UI"
    After installing or updating, reload your UI with:
    ```
    /reload
    ```

If you continue to experience issues, please [report them on GitHub](https://github.com/OsulivanAB/SpectrumFederation/issues).
