# WowUp Installation

WowUp is a modern, open-source addon manager for World of Warcraft.

## Prerequisites

1. Download and install [WowUp](https://wowup.io/)
2. Launch WowUp and let it detect your WoW installation

## Installation Steps

### Step 1: Search for the Addon

1. Open WowUp
2. Click the **Get Addons** tab
3. Search for "Spectrum Federation"

### Step 2: Install

1. Click the **Install** button next to Spectrum Federation
2. Wait for the download to complete
3. The addon will be automatically installed to the correct directory

### Step 3: Verify Installation

1. Launch World of Warcraft
2. At the character selection screen, click **AddOns**
3. Verify that "Spectrum Federation" is listed and enabled
4. Enter the game and check for the load message in chat

## Updating

WowUp automatically checks for updates:

1. When you launch WowUp, it will scan for available updates
2. Click **Update All** or update Spectrum Federation individually
3. Updates are installed automatically

## Beta Versions

To receive beta updates:

1. In WowUp, go to **Options**
2. Enable **Show Beta Versions** or similar setting
3. Beta releases will now appear in the update list

!!! warning "Beta Releases"
    Beta versions may contain experimental features and bugs. Use at your own risk!

## Uninstalling

To remove the addon:

1. In WowUp, find Spectrum Federation in **My Addons**
2. Click the **Uninstall** button
3. The addon will be removed from your WoW directory
