### Manual Installation

1. Copy the `SpectrumFederation/` folder into your WoW installation directory:
   ```
   World of Warcraft\_retail_\Interface\AddOns\SpectrumFederation\
   ```
   or
   ```
   World of Warcraft\_beta_\Interface\AddOns\SpectrumFederation\
   ```

2. Restart WoW or reload your UI with `/reload`

3. The addon should print a success message in chat when you log in
